CAT Active General FAQs

Q1. What types of services does CAT offer?

Ans. It provides a platform to check if application is Cloudable or not. It also helps you to know Migration Pattern and Cloud Provider for the application.

Q2. Who can use the application?

Ans. If anyone wants to Migrate there application to cloud then they can use the application to know if their application is cloudable or not.

Q3. How to get access to the application, do I need to create an account?

Ans. No need to create an account. First of all you have to contact the owner or the CAT application. Then they will create a new user(account) on behalf of you and will provide credential. Now using those credential you can access to the application.

Q4. As a client can I add any user or create user if yes then how can i create a new user?

Ans. Definitely you can create users for accessing your applications. You just need to go to users console where you can find Add User button. Just click on that you can add new user by providing the user information.

Q5. How to update user or delete user as per need? Is it possible?

Ans. Yes, it is possible to delete and update user. In user console there is a button for deleting and updating users. By clicking on Delete button you can delete a user. By clicking on Update User button a new window will appear by providing user information you can update it.

Q6. Can I import user if I have a list of users information in my excel file?

Yes, we have provided import user functionality in the application. Import User button is present on users console. After clicking on the button you can import users. If you do not have the CSV file, we have provided a sample CSV in the application. You can fill the file and import it as well.

Q7. How can I export users list?

Ans. You just need to go to users console and click on Export Users button. It will download a CSV file containing all the users information.

Q8. Is it a configurable application? If yes how it works?

Ans. Yes, it is a configurable application. You can add any number of Users, Questions, Applications to it. As per your requirement set it and will check the cloudability, Migration Pattern, Cloud Provider for the application.

Q9. How can I assist an application?

Ans. For Assisting application go to application console then under action column assist button is present. After clicking on the button you can give answer for your application.

Q10. How to add questions?

Ans. To add questions go to the assessment question console then click on Add Questions button to add questions. you can add answers for your questions here. you can also select the question type here.

Q11. Can the application have different clients?

Ans. Yes we have different clients and different questions as well as different set of rules for them.

Q12. Can I deactivate users questions and application?

Ans. Yes we are providing buttons in respective pages for this purpose. For this you have to go to respective console and under action column deactivate button is present. Click on those button to make them invisible from UI.

Q12. Can I export PDF file reports for my application?

Ans. Yes, you can. In the console go to report section. There you can find button for exporting PDF files for your application.

Q13. What is Decision Tree. How it works?

Ans. It contains the rules that is sets to check Cloudability, Migration Pattern, Cloud Provider. You can set rules here. Based on the rules it will provide you results.

Q14. How to set rules for Cloudability check?

Ans. Under Decision Tree module Cloudable module is present. After clicking on cloudable module all the cloudable questions will appear. There you can provide the rules for cloudable.

Q15. What all type of Migration Pattern are there?

Ans. Public PAAS, Replatform, ReHost

Q16. How to set rules for different migration pattern?

Ans. Under decision tree Migration Pattern module is present. There you have to select for which migration pattern you want to add rules. After select that a new page will appear where it will contain questions and you have to provide rules for them.

Q17. What all type of Cloud Providers are there?

Ans. There are different type of Cloud Provider are present some of them are AWS, AZURE, Openshift, Global IT cloud. Cloud Provider decides to which cloud you can move you application.
